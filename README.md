# Global Financial Simulation

Agent-based global financial system simulation exploring geopolitical scenarios, trade dynamics, and monetary policy interactions.

## Key Findings

- TDA (Topological Data Analysis) reveals persistent structural patterns in simulated financial networks
- China-specific political economy dynamics modeled with high fidelity
- Scenario analysis for trade war escalation, currency decoupling, and sanctions regimes

## Structure

- `src/` — Simulation engine and agent definitions
- `data/` — Input data and calibration parameters
- `docs/` — Documentation and data format specifications
- `analysis/` — Post-simulation analysis tools
